# Problem

---

**1. 세분화되는 여행취향**
세분화 시대의 밀레니얼 세대들은 남들과 똑같은 관광지투어가 아닌, 자신만의 가치관과 취향에 맞는 맞춤여행을 원합니다. 

**2. 대면 여행가이드의 쇠퇴**
코로나19로 인해 대면 여행가이드가 없어지고, 사람들은 사람이 붐비는 관광지를 피하려고 합니다.

**3. 여행 과정의 간편함 추구**
시간을 재화로 여기는 밀레니얼 세대들은 여행계획부터 예약과정까지 모두 간편함을 추구합니다.
